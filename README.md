# 🏆 Site de Palpites da Copa do Mundo 2026™ (16-Avos de Final)

Plataforma interativa de bolão e palpites para os jogos dos 16-avos de final da Copa do Mundo de 2026!

## ⚽ Funcionalidades
- **Mural ao Vivo:** Acompanhe os palpites da torcida em tempo real.
- **Armazenamento Híbrido e Conectado (Cloud + Local):** Funciona perfeitamente em hospedagens estáticas como o **GitHub Pages**, salvando e sincronizando palpites em nuvem online.
- **Regras Oficiais do Jogo:** Limite de 2 palpites por torcedor para cada partida e bloqueio automático de envios no início do jogo (considerando fuso horário de Cuiabá -1h).
- **Design FIFA PlayZone:** Visual moderno, responsivo para smartphones e tablets, com bandeiras oficiais e modo noturno de alto contraste.

## 🚀 Como Executar
1. **Pela Web:** Acesse diretamente no GitHub Pages: [https://olyrum.github.io/jogocopa/](https://olyrum.github.io/jogocopa/)
2. **Localmente:**
   ```bash
   npm install
   node server.js
   ```
   Acesse `http://localhost:3050`.
